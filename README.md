#### dedecmsLayui

1.  dedecms5.7的后台源码，layui的版本我更新到了最新版2.5.6
2.  后台菜单我注释了一些不用的功能，自己可以打开，在后台index_body.html里面修改即可
3.  所有的后台页面以前点击单选按钮和多选按钮的时候必须点击选框，现在只要点击文字就可以选中了
